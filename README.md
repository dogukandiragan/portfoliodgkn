# PortfolioDgkn
A multi-layered portfolio tracking app built with ASP.NET Core MVC & Web API, featuring JWT authentication, Redis caching, and real-time price integration. 

🚀 Live Demo <a href="http://portfolio.dgkn.net" target="_blank">Projeyi canlı görmek için tıklayın</a>

  - ASP.NET Core MVC
  - Web API
  - N-Tier (Core / DataAccess / BusinessLogic / Presentation)
  - Entity Framework Core
  - ASP.NET Identity
  - JWT Token Auth                                                                                                                                              
  - Custom Middleware (exception handler, status code)
  - Redis (IDistributedCache)
  - IHttpClientFactory (named clients)
